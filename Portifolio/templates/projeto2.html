Este projeto visa criar uma solução inteligente para o abastecimento automático de água para pets. Desenvolvido com a IDE Arduino 2.0 e o microcontrolador ESP32, o sistema oferece uma plataforma interativa que permite ao cliente monitorar e gerenciar o nível de água do seu pet de maneira eficiente. A integração com a Arduino Cloud possibilita a criação de dashboards personalizados, permitindo a configuração de horários e o monitoramento remoto do abastecimento. Com essa solução, o cliente pode assegurar que seu pet tenha sempre água fresca disponível, sem necessidade de intervenção manual constante.

IDE Utilizada

Neste projeto, utilizou-se a IDE Arduino 2.0 para desenvolver a lógica e o código.

Dashboard Botões
