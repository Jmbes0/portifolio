Desenvolvi uma solução de web scraping para o site do TJSP, destinada a um cliente advogado que precisava automatizar a tarefa repetitiva de baixar processos. O software opera de maneira intuitiva: o usuário insere o nome do foro (cidade) e o CNPJ (banco) para a pesquisa, e o sistema realiza o download de todos os processos relevantes conforme solicitado. A execução é feita em segundo plano, permitindo que o computador seja utilizado para outras tarefas simultaneamente. Após a conclusão do processo, o software fecha automaticamente e armazena todos os links baixados, prevenindo a duplicação de downloads em pesquisas futuras.

Ide de desenvolvimento

O desenvolvimento foi realizado utilizando o Visual Studio Code.

Janela de Dados

Janela para inserir os dados do foro e da cidade onde será realizada a busca.

Pasta para de downloads

Esta é a pasta onde todos os PDFs baixados serão armazenados.

Links

Neste arquivo .txt são armazenados todos os links já baixados, evitando a duplicação de downloads.
