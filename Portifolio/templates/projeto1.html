O projeto foi desenvolvido para um cliente que necessitava de monitoramento de gás e controle remoto de dispositivos em sua residência, como ar-condicionado e umidificador de ar. Utilizamos o microcontrolador ESP32 por sua robustez e capacidade de atender às exigências do projeto. O sistema opera de forma remota por meio de um dashboard na Arduino Cloud, onde o cliente pode definir a temperatura e a umidade desejadas. Além disso, o dashboard permite o controle total dos dispositivos conectados e a visualização da temperatura e umidade do local a partir de sensores de monitoramento em tempo real. Em relação à segurança, o projeto conta com um botão de alerta para vazamento de gás. Caso o sensor detecte um vazamento, o botão é acionado, gerando um alerta no dashboard e enviando uma notificação ao cliente via e-mail ou mensagem push. Simultaneamente, uma válvula solenoide é ativada automaticamente para interromper o fornecimento de gás, prevenindo possíveis danos à residência. Este mesmo sistema pode ser adaptado para uso industrial. No exemplo, utilizamos um buzzer para simular uma sirene, oferecendo um alerta sonoro em caso de vazamento de gás, o que amplia a segurança em ambientes industriais.

Ide de desenvolvimento

Utilizei a IDE Arduino 2.0 para o desenvolvimento deste projeto.

Dashboard do projeto

Dashboard na Arduino Cloud para controle e monitoramento do projeto.

Mensagem De Alerta

Mensagem de alerta por e-mail (ou mensagem push) informando sobre o acionamento da válvula de gás.
