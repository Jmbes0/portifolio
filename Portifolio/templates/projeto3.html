O projeto foi desenvolvido para monitorar a pressão exercida pelos galhos de árvores. Esse monitoramento permite prever a perda de água nos galhos, oferecendo uma estimativa precisa de quanto tempo, após o corte das árvores, elas permanecem saudáveis para o transporte. Com isso, é possível evitar possíveis perdas de mercadorias. A solução foi personalizada para um cliente específico, garantindo que suas necessidades fossem atendidas com precisão.

IDE Arduino Cloud

Neste projeto, a IDE Arduino Cloud foi utilizada para desenvolver a lógica e a programação do sistema.

Gráfico de Pressão e Altitude

O botão inicia o monitoramento, enquanto os gráficos exibem a pressão e a altitude.

Gráfico de Temperatura

Gráficos para a temperatura ambiente e para a média da pressão em relação à altitude.
